Research Pipeline

Description of the Exposed work

  • Dataset

    4,266 images from 91 well known painters, such as Claude Monet, Frida Kahlo, Henri Matisse, Jackson Pollock, Picasso, Rembrandt, and Salvador Dali.

    Source of Dataset: Site , Article

  • Compute Average Author Complexity

    Study average author complexity using NC and NBDM for different quantization levels (4, 6 and 8 bits)

  • Fingerprint of the authors' complexity.

    Compute regional NC (64 Regions) for each of the paintings. Afterwards, compute for each author their average regional complexity.

  • Phylogenetic Tree

    Using each authors' average regional complexity, we computed a distance matrix and used it to create e phylogenetic tree that relates each author.

Average Complexity

Average values of Normalized Compression (NC) and Normalized Block Decomposition method (NBDM) for each author when images were quantitized for 4, 6 and 8 bits.

ANC
Average Normalized Compression (ANC) for each author where images of paintings where quantitized for 4, 6 and 8 bits. The authors are sort given the value of ANC.
NBDM1
Average Normalized Block Decomposition Method (ANBDM1) for each author where images of paintings where quantitized for 4, 6 and 8 bits. The authors are sort given the value of ABDM.
NBDM2
Average Normalized Block Decomposition Method (ANBDM2) for each author where images of paintings where quantitized for 4, 6 and 8 bits. The authors are sort given the value of ABDM.
